# 🌍 Global Commodity Markets Analysis (2000–2026)

## 📌 Project Overview
A 25-year analysis of global commodity prices covering Gold, Silver, 
Platinum, Crude Oil WTI, Copper, and Natural Gas from 2000 to 2026.

## 📊 Live Dashboard
🔗 [View Interactive Tableau Dashboard](https://public.tableau.com/shared/JM7YXJWB9)

## 🔍 Key Insights
- Gold surged 400%+ post-2020 amid global uncertainty
- Platinum peaked during the 2008 Financial Crisis
- Volatility spiked sharply during COVID-19 (2020) and Gold ATH (2025)
- Natural Gas and Copper remained relatively stable over 25 years

## 🛠️ Tools Used
- Tableau Public — Dashboard & Visualisation
- Microsoft Excel — Data Cleaning & Preparation
